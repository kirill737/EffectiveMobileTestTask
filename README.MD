# Сериализация и десериализация двусвязного списка

## Описание функция
В данной проекте были реализованы 4 функции:
1. ```serializeToText``` - Сериализация двусвязного списка в текстовое представление
2. ```deserializeFromText``` - Десериализация двусвязного списка из текстового представления
3. ```serializeToBin``` - Сериализация двусвязного списка в бинароное представление
4. ```deserializeFromText``` - Десериализация двусвязного списка из бинароное представления

## Запуск
Для запуска можно ввести следующие команды:
```
g++ main.cpp -o run
./run.exe
```

## Тесты
В качестве тестов была выбрана следующая последовательность действий:
1. Десериализация списка из текстового файла "inlet.in"
2. Сериализация полученного списка в бинарный файл "outlet.out"
3. Десериализация списка из бинарного файла "outlet.out"
4. Сериализация полученного списка в текстовый файл "result.txt"

По завершению которой данные из файлов "inlet.in" и "result.txt" должны совпадать.
